# Análisis de Regresión y Selección de Variables (R)

## Descripción del Proyecto
Proyecto de modelización estadística centrado en la identificación de las variables explicativas más relevantes y en la construcción de modelos de regresión interpretables a partir de datos reales.

## Objetivo
Explicar el comportamiento de una variable respuesta mediante un conjunto reducido de predictores decisivos, manteniendo la mayor parte del poder explicativo del modelo completo.

## Metodología
- Análisis exploratorio de datos y estudio de correlaciones
- Ajuste de modelos de regresión lineal múltiple
- Diagnóstico de multicolinealidad
- Aplicación de Análisis de Componentes Principales (PCA) como apoyo a la selección de variables
- Evaluación del modelo priorizando interpretabilidad y robustez frente a complejidad innecesaria

## Resultados
- Identificación de las variables explicativas más influyentes
- Reducción del modelo final a dos o tres predictores clave
- Interpretación clara de los efectos y relaciones entre variables

## Herramientas
- R
- Librerías de análisis estadístico y visualización

## Conclusiones Clave
- Modelos más simples pueden ofrecer un poder explicativo comparable con mayor interpretabilidad
- La selección de variables es fundamental para modelos orientados a la toma de decisiones
- El uso de diagnósticos estadísticos es esencial para garantizar la fiabilidad del modelo

## Nota
Este proyecto fue desarrollado en el contexto de una asignatura de estadística y se presenta como un ejemplo de razonamiento estadístico aplicado y análisis orientado a la toma de decisiones.
